# Black-Scholes Model with Greeks

This project implements the 'Black-Scholes (BS) Model in Python using an object-oriented approach. It calculates European option prices (Call/Put) and provides a full analysis of Option Greeks (Delta, Gamma, Vega, Theta, Rho).
